Over Mij

Mijn naam is Mika van de Kuit en ik ben 20 jaar. Ik volg de opleiding software developer MBO niveau 4 op ROC van Flevoland. Ik heb deze opleiding gekozen omdat ik al enige ervaring had met programmeren en game development (Scratch, GameMaker 2), en omdat ik alles wilde leren over het maken van games. De opleiding bevalt mij heel goed.

Snake - Java

In mijn eerste jaar op ROCvF ben ik bezig geweest met processing, een graphics library voor Java. Ik heb snake hierin nagemaakt. Het is compleet met screen-wrapping en een schakelbare AI die automatisch naar het fruit gaat. Dit project laat zien dat ik al heel vroeg de probleemoplossende vaardigheden had die in software development nodig zijn.

Yahtzee - JavaScript

Ik heb op ROCvF ook Yahtzee gemaakt. Ik kreeg de layout van de pagina, en de opdracht was om alle logica achter het spel te programmeren in JavaScript. Dit vond ik echt een super leuke opdracht. Dit is ook hier online speelbaar.

Minecraft Mods

In mijn vrije tijd ben ik bezig geweest met minecraft mods maken. Ik heb veel minecraft met mods gespeeld over de jaren heen, en ik vind het super leuk om zelf mods te kunnen maken.
